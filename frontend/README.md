# CampusSync - Premium Campus Resource Control Center

A modern, high-performance SaaS dashboard for managing institutional resources with real-time governance.

##  Tech Stack
- **Frontend:** React + Vite + TypeScript
- **Styling:** Tailwind CSS (Modern Dark Mode)
- **Animations:** Framer Motion
- **State:** Zustand
- **Real-time:** Socket.io-client
- **Charts:** Recharts

##  Getting Started

### Prerequisites
- Node.js & npm
- Backend server running (see `../backend/README.md`)

### Installation
```bash
cd frontend
npm install
```

### Development
```bash
npm run dev
```

##  Features
- **High-Conversion Landing Page:** Animated product preview and feature grid.
- **Smart Dashboard:** KPI metrics, usage analytics, and recent activity feed.
- **Conflict-Safe Booking:** Real-time overlap detection during resource scheduling.
- **Role-Based Workflows:** Distinct permissions for Students, Faculty, and Admins.
- **Immutable Audit Logs:** Full traceability for every system action.
- **Real-time Notifications:** Instant WebSocket alerts for booking status changes.
